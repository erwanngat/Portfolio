Projet site web

Site internet de vente de chaussure

Ce projet à été fait en PHP 8.2.0 sur le framework Laravel v10.34.2 et le framework Tailwind CSS v3.3.6

    Il contient:


  • Un CRUD
  • Une APi pour un CRUD
  • Une gestion des utilisateurs avec middleware
  • Des composants livewire pour les fonctionnalités interactives
  • Une validation des données pour garantir une précision
  • Une gestion des migrations et des seeders
  • Une gestion du panier
  • Une gestion des tailles des chaussures
  • Une gestion des stocks
